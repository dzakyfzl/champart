# CHAMPART

## Menjalankan via docker 
1. install my sql buat password dna segala macem 
2. buat file **.env** di root folder sejajar dengan file docker-compose.yml dengan yang sama pada file **example.env**

3. buka app docker 
4. buka terminal di root jalankan perintah `$ docker-compose down -v` 
4. kemudian jalankan perintah `$ docker-compose -f docker-compose.yml up --build` 
5. buka Kembali docker dan lihat containernya cari bagian be champart dan klik yang 8000:8000 atau bisa ke local host/8000
